Web Zip Version 1.0

Rest API Reference

All Rest API calls need to provide a service token HTTP header. x-webzip-api-key:value

Token values are stored in META-INF/servicetoken.json

Configuration of folder to store archives on the server can be done in: META-INF/webzip.properties
