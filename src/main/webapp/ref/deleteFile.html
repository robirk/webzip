Delete File or Folder

Method

DELETE /rest/archive/{archivename}/content?file={file_or_directory}

Path Parameters

Name Description
archivename The name of the ZIP archive.

Query Parameters

Name Description
file Name and path of file or directory

Result

A JSON object containing confirmation or error status.

Name Value Description
status String Value "ok" or "error"
filename String If status "ok", file or directory name.
message String If status "error", description of the error.

Response Codes

Code Media Type Description
200 application/json Request successful.

Example

DELETE http://localhost:8084/webzip/rest/archive/backup.zip/content?file=Project_SCM%203.0%2Fnew.json
