Download Archive

Method

GET/rest/archive/{archivename}

Path Parameters

Name Description
archivename The name of the ZIP archive.

Result

Binary data of archive.

Content type: application/octet-stream

Response Codes

Code Media Type Description
200 application/octet-stream Request successful.
404  text/html File not found.

Example

GET http://localhost:8084/webzip/rest/archive/backup.zip
