Download File

Method

GET/rest/archive/{archivename}/content?file={file}

Path Parameters

Name Description
archivename The name of the ZIP archive.

Query Parameters

Name Description
file

Name of the file

Result

Binary data of file.

Content type: application/octet-stream

Response Codes

Code Media Type Description
200 application/octet-stream Request successful.
404 text/html File not found.

Example

GET http://localhost:8084/webzip/rest/archive/backup.zip/content?file=Project_SCM%203.0%2Fnew.json
