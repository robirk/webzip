List Files

List all folder/files of an archive.

Method

GET /rest/archive/{archivename}/content

Path Parameters

Name Description
archivename The name of the ZIP archive.

Result

A JSON array of files.

Name Value Description
archive String Archive name
content Array Array of files
path String Path of file location within archive
name String File name
length Integer File size
lastModified Date Last modified date
type String Values: file or folder
